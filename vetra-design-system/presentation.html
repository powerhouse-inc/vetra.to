Getting Started

  1. Clone the repository and install dependencies using the package manager of your choice.
  2. Configure your environment variables and set up the local development server.
  3. Import the design tokens and connect your Figma library for live synchronization.
  4. Build your first component using the provided templates and submit a pull request.

Simple API

Get up and running with just a few lines of code. Our SDK handles authentication, caching, and error recovery automatically.
